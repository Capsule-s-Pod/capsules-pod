Getting Started

Joining Capsule's Pod is easy for both Java and Bedrock players. Follow the instructions below to get started:

For Java Edition Players

  1. Open the Multiplayer tab.
  2. Select "Add Server."
  3. Set the server name to anything you like.
  4. Enter the server address: capsules-pod.serveminecraft.net:25565

For Bedrock Edition Players

  1. Open the Servers tab.
  2. Select "Add Server."
  3. Set the server name to anything you like.
  4. Enter the server address: capsules-pod.serveminecraft.net

About Capsule's Pod

Capsule's Pod is a simple Minecraft server run on a spare laptop under a docker container, using Geyser and Floodgate to assist in running the proxy connection for Bedrock players.
